By Splizard with help from Temperest.
Also thanks to ironzorg for creating hatches which this mod is based on.

INSTALL:
----------
If you have the mesecons (jeija) mod installed, place ./mesecons/gates 
in your mods folder otherwise place ./gates in your mods folder.

USAGE:
-------
Open a gate by clicking on it. 
You can stack the gates to create doors that open with a single click.
They can also be used as a window that opens.
